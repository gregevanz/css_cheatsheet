CSS Selectors Cheatsheet

CSS Selector Description Example
Type Used to match all elements of a given type or tag name p {}
Class Matches elements based on the contents of their class attribute .some-class {}
HTML attributes with Multi-values We can have multiple attribute values, they are separated via a space div class="value1 value2 value3"
CSS ID Selectors Matches elements based on the contents of their id attribute. ID values should be unique #some-id{}
Class and ID Selectors Class selectors tend to be reused (prepended with a '.') while ID selectors tend to be unique (prepended with a #) div class="value1 value2 value3"
